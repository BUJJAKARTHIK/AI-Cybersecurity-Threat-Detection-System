# 🔐 AI-Powered Cybersecurity Threat Detection System

## 📌 Overview
This project uses Machine Learning to detect cyber attacks from network traffic data. It simulates real-world cybersecurity environments used in SOC (Security Operations Centers).

---

## 🚀 Features
- Intrusion detection system
- Machine learning-based classification
- Confusion matrix visualization
- Threat detection alerts
- Streamlit dashboard UI

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Streamlit

---

## 📊 Dataset
NSL-KDD Dataset (Network Intrusion Detection dataset)

---

## ⚙️ How to Run

### Install dependencies:

pip install -r requirements.txt


### Run model:
~~~ bash
python main.py
~~~

### Run dashboard:
~~~ bash
streamlit run app.py
~~~
